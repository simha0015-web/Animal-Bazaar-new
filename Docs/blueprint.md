# **App Name**: PashuMandi

## Core Features:

- User Authentication: Secure user login and registration using email/password and phone OTP.
- Animal Posting: Users can create listings for animals with details such as type, breed, price, and location.
- Image Upload: Users can upload images of their animals to showcase in the listings.
- Search and Filtering: Users can search for animals by name, breed, and location, and filter results.
- Featured Listings: Highlight featured animal listings on the home screen.
- Real-time Database: Store the listings in the firebase Realtime database
- Listing summarization: Use a generative AI tool to generate a listing summary and suggest similar items. 

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke nature and freshness.
- Background color: Light green (#F1F8E9), a desaturated version of the primary color, for a calm backdrop.
- Accent color: Earthy orange (#E65100) for highlights and calls to action.
- Body and headline font: 'PT Sans', a modern and friendly sans-serif.
- Use farm-themed icons to represent different animal categories.
- Clean, card-based layout with clear separation of content sections.
- Subtle transitions and animations for a smooth user experience.
